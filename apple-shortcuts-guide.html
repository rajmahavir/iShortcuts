Welcome to Apple Shortcuts

Shortcuts is a powerful automation app for iOS and iPadOS that lets you create custom workflows to streamline your daily tasks. With Shortcuts, you can automate actions across multiple apps with just a tap or by using Siri.

What You Can Do with Shortcuts

  • Automate everyday tasks like sending messages, playing music, or getting directions
  • Create custom workflows that work across multiple apps
  • Use Siri to run shortcuts with your voice
  • Set up automations that trigger based on time, location, or other conditions
  • Share shortcuts with friends and family

Getting Started with Shortcuts

Opening the Shortcuts App

The Shortcuts app comes pre-installed on iOS 13 and later. To open it:

  1. Find the Shortcuts app on your Home Screen (it has a white icon with colorful squares)
  2. Tap to open the app
  3. You'll see the Gallery tab with pre-made shortcuts you can add

Your First Shortcut

The easiest way to get started is by adding a shortcut from the Gallery:

  1. Open the Shortcuts app
  2. Tap the Gallery tab at the bottom
  3. Browse through the available shortcuts
  4. Tap Add Shortcut on one that interests you
  5. The shortcut is now in your My Shortcuts tab

Create a New Shortcut

Building Your First Custom Shortcut

Creating your own shortcut is simple:

  1. Open the Shortcuts app
  2. Tap the My Shortcuts tab
  3. Tap the + button in the top right
  4. Tap Add Action
  5. Search for or browse actions you want to include
  6. Tap actions to add them to your shortcut
  7. Arrange actions in the order you want them to run
  8. Tap Done when finished

Understanding Actions

Actions are the building blocks of shortcuts. Each action performs a specific task, such as:

  • Get Current Weather - Retrieves weather information
  • Send Message - Sends a text message
  • Play Music - Plays music from Apple Music
  • Get Directions - Opens Maps with directions
  • Set Timer - Creates a timer
üìù Note: Actions can pass data to each other. The output of one action can become the input for the next action in your shortcut.

Running Your Shortcuts

Ways to Run a Shortcut

There are several ways to run your shortcuts:

1. From the Shortcuts App

Simply tap the shortcut in your My Shortcuts tab.

2. Using Siri

You can run any shortcut by asking Siri. Just say "Hey Siri" followed by the name of your shortcut.

3. From the Home Screen

Add shortcuts to your Home Screen for quick access:

  1. Open the Shortcuts app
  2. Long-press on a shortcut
  3. Tap Share
  4. Tap Add to Home Screen
  5. Customize the name and icon if desired
  6. Tap Add

4. From the Share Sheet

Some shortcuts can run from the Share Sheet in other apps.

5. Using Widgets

Add a Shortcuts widget to your Home Screen or Today View for quick access to your favorite shortcuts.

Personal Automations

What are Automations?

Automations let you run shortcuts automatically based on triggers like:

  • Time of Day - Run shortcuts at specific times
  • Alarm - Trigger when you stop an alarm
  • Arrive/Leave - Based on your location
  • Before I Commute - Before you head to work or home
  • Email - When you receive email from specific people
  • App - When you open or close specific apps
  • Airplane Mode - When you turn airplane mode on/off
  • Do Not Disturb - When DND changes

Creating an Automation

  1. Open the Shortcuts app
  2. Tap the Automation tab
  3. Tap + to create a new automation
  4. Choose a trigger (when you want it to run)
  5. Configure the trigger conditions
  6. Tap Next
  7. Add actions for what you want to happen
  8. Choose whether to run automatically or ask before running
  9. Tap Done

Advanced Features

Variables and Magic Variables

Variables let you store and reuse information in your shortcuts:

  • Use Set Variable to store a value
  • Use Get Variable to retrieve the value later
  • Magic Variables automatically store output from actions

Conditional Logic

Add decision-making to your shortcuts with If statements:

  • Run different actions based on conditions
  • Create more intelligent, adaptive shortcuts
  • Handle different scenarios automatically

Repeat Actions

Use Repeat actions to run the same actions multiple times:

  • Repeat - Run actions a specific number of times
  • Repeat with Each - Run actions for each item in a list

Scripting

For advanced users, shortcuts support:

  • JavaScript code
  • Shell scripts (on Mac)
  • Python scripts (on Mac)
  • Dictionary and JSON manipulation
  • Regular expressions

Sharing Shortcuts

Share Your Shortcuts

You can share shortcuts with others:

  1. Open the shortcut you want to share
  2. Tap the Share button
  3. Choose how to share (Messages, AirDrop, etc.)
  4. The recipient can add it to their library

iCloud Link

Create an iCloud link to share shortcuts publicly:

  1. Tap the Share button on a shortcut
  2. Tap Copy iCloud Link
  3. Share the link anywhere (social media, website, etc.)
üìù Note: When someone downloads your shortcut, they'll see which actions it contains and can review it before adding it to their library.

Troubleshooting

Common Issues and Solutions

Shortcut Won't Run

  • Check that all required permissions are granted
  • Make sure the actions are in the correct order
  • Verify that actions have valid inputs
  • Check for any error messages

Automation Not Triggering

  • Verify the automation is enabled
  • Check location permissions (for location-based automations)
  • Ensure Low Power Mode isn't preventing background triggers
  • Check that the trigger conditions are met

Siri Can't Find Shortcut

  • Make sure the shortcut name is unique and easy to pronounce
  • Try re-adding the Siri phrase
  • Check that Siri has permission to access Shortcuts

Action Not Working

  • Check that the app the action uses is installed
  • Verify app permissions are granted
  • Update to the latest version of the app
  • Try recreating the action
